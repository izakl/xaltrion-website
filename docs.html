Documentation

Quick reference for getting Xaltrion AI installed and running locally.

Installation

Xaltrion ships as a Windows installer. macOS and Linux are on the roadmap.

  1. Download the latest .exe installer from the Download page.
  2. Run the installer. Windows SmartScreen may warn you — click More info then Run anyway. The installer is unsigned during beta.
  3. On first launch, the 9-step setup wizard opens automatically.
  4. The wizard guides you through: licence activation → notifications → brokers → data feeds → AI keys → trading profile → view selection.
  5. After completing the wizard, click Start trading — you're done.

Broker setup

Brokers are configured in the wizard (Step 4) or anytime via Settings → Brokers.

Alpaca (US stocks & crypto)

Sign up at alpaca.markets and generate API keys. Supports three modes: Paper only (simulated trading), Live only (real money), or Both.

Luno (ZAR crypto)

Create API keys at luno.com under Account → API keys. Grant read and trade permissions.

VALR (ZAR crypto)

Generate keys in your VALR account under API management. Grant read and trade permissions.

Coming soon

Binance, Bybit, OKX, Rain, Interactive Brokers, MetaTrader 5, and more. See the full roadmap.

Data feeds

Data feeds are configured in the wizard (Step 5) or anytime via Settings → Data Feeds. More feeds give the AI richer market context.

Massive

Real-time prices, news, and crypto and stock data. The individual plan works well for crypto trading; upgrade for real-time stocks and options data. Get a key at massive.com.

Finnhub

Sentiment scoring, earnings calendar, analyst recommendations, and social sentiment. A free API key is available at finnhub.io.

FRED

US macroeconomic data — interest rates, inflation, GDP. Free API key available from the St. Louis Fed.

CoinGecko

Global crypto prices and market data. No API key needed — enabled by default. Xaltrion's usage stays well within the free tier.

AI setup

AI keys are configured in the wizard (Step 6) or anytime via Settings → AI.

Claude API key (required)

Powers all trading decisions — evaluates signals, explains reasoning, and manages risk. Get a key at console.anthropic.com. Without this key Xaltrion cannot make AI-powered decisions.

Perplexity API key (optional)

Adds real-time web market context on top of your data feeds. Get a key at perplexity.ai. Without it Xaltrion runs in Claude-only mode, which is still fully functional.

Model selection

Choose your Claude model in Settings → AI:

  • Claude Haiku 4.5 — faster, more cost-effective. Recommended for high-frequency trading.
  • Claude Sonnet 4.6 — more capable, higher cost. Recommended for complex multi-asset strategies.

Dashboard guide

The portal opens at http://localhost:8080 after starting the app. Choose your view from the top ribbon.

  • Simple — equity curve, open positions, recent decisions. Clean and calm.
  • Operator — multi-broker portfolio, AI spend tile, risk guardrail bars, equity graph with range selector.
  • Expert — full console, AI reasoning log, live feeds, orders, backtest, heartbeat.
  • Settings — brokers, data feeds, AI provider, risk limits, notifications, licensing.

Troubleshooting

Windows SmartScreen blocks installer

Click More info then Run anyway. The installer is unsigned during beta.

Setup wizard not appearing on first launch

Go to Settings → General → Resume onboarding to restart the setup wizard.

License token cache stale

Go to Settings → System → Clear activation cache, or delete .activation_cache.json and restart Xaltrion.

Machine binding mismatch

If you moved to a new machine, contact support@xaltrion.ai to clear the binding.

Port 8080 already in use

Close any other app using port 8080 and restart Xaltrion.

Perplexity not working

Check your API key is saved in Settings → AI → Perplexity. Without a valid key, Xaltrion runs in Claude-only mode.

Reset portal layout

Use Settings → System → Reset portal layout to restore the default layout.

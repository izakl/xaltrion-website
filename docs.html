Documentation

Quick reference for getting Xaltrion installed and running locally.

Installation

Xaltrion ships as a Windows installer. macOS and Linux are on the roadmap.

  1. Download the latest .exe installer from the Download page.
  2. Run the installer. Windows SmartScreen may warn you — click More info then Run anyway. The installer is unsigned during beta.
  3. On first launch, the 9-step setup wizard opens automatically.
  4. The wizard guides you through: licence activation → notifications → brokers → data feeds → AI keys → trading profile → view selection.
  5. After completing the wizard, click Start trading — you're done.

Broker setup

Brokers are configured in the wizard (Step 4) or anytime via Settings → Brokers. Pick a broker for sign-up details:

Each broker runs in one of three modes, switchable per broker: Demo (the broker’s own paper/demo account, where one exists), Simulated (the app simulates fills against real live prices — run a live-keyed account with a virtual balance in your choice of currency, no separate demo needed), and Live (real money — off by default, gated behind a typed “ENABLE LIVE” confirmation). The AI behaves identically in every mode; only execution differs.

Alpaca LIVE Stocks · ETFs · Crypto · [US]

US stocks, ETFs and crypto, commission-free. Three lanes: Paper (simulated), Live (real money), or both — paper is included with every Alpaca account.

Sign up & create API keys: alpaca.markets → Home → API Keys.

Luno LIVE Crypto · [ZAR] · [SA]

South African crypto exchange — ZAR markets (BTC, ETH and major pairs).

Create keys at luno.com → Account → API keys. Grant read and trade permissions.

VALR LIVE Crypto · [ZAR] · [SA]

SA crypto exchange with more pairs and deeper liquidity. ZAR markets.

Create keys at valr.com → API Management. Grant read and trade permissions.

Kraken BETA · SIM / LIVE Crypto · [Global]

Global crypto spot (1,400+ pairs). Kraken runs the full AI strategy in Simulated mode (fills against live prices on a virtual balance) or Live (real money — off by default, typed “ENABLE LIVE” confirmation). Kraken has no native demo, so use Simulated to trial it risk-free.

Create keys at kraken.com → Settings → API.

Kraken Futures BETA · DEMO / SIM / LIVE Perps · [Global]

Crypto perpetual futures. Run it on the broker-native demo, in Simulated mode (live prices, virtual balance), or Live (real money — off by default, typed confirmation; live stays long-only, 1× and reduce-only for safety).

Demo + live keys are generated separately inside the Kraken Futures / demo platforms.

IG BETA · DEMO / SIM / LIVE CFD · Forex · [Global]

CFDs on indices, commodities, forex and shares. Run it on the IG demo, in Simulated mode (live prices, virtual balance), or Live (real money — off by default, typed confirmation; live needs an activated IG account).

Get API credentials at ig.com (My IG → Settings → API). Demo and live use separate credentials.

HitBTC BETA · SIMULATED Crypto · [Global]

Crypto spot & futures. Runs in Simulated mode — the AI trades against live HitBTC prices on a virtual balance you set, so you can trial it with zero risk.

Create API keys at hitbtc.com (HTTP Basic API key + secret).

Data feeds

Data feeds are configured in the wizard (Step 5) or anytime via Settings → Data Feeds. They give the AI richer market context — more signal, better decisions.

None of these are required. The bot runs on your broker’s live prices plus your Claude key. CoinGecko (free, on by default) already covers crypto prices; add the others that match what you trade. Pick a feed for details:

CoinGecko Recommended Free · no key

Provides: global crypto prices and market data. On by default — no API key needed; usage stays well within the free tier.

Massive Recommended Free tier · paid upgrade

Provides: real-time prices, news, and crypto + stock data. The individual plan works well for crypto; upgrade for real-time stocks and options. Most useful if you trade stocks.

Get a key: massive.com.

Finnhub Optional Free key

Provides: sentiment scoring, earnings calendar, analyst recommendations, and social sentiment fed into the AI’s context.

Free key: finnhub.io.

FMP (Financial Modeling Prep) Optional Free tier

Provides: the upcoming-earnings calendar, so the AI is aware of earnings events. Free tier is sufficient (cached aggressively).

Get a key: financialmodelingprep.com.

FRED Optional Free key

Provides: US macroeconomic data — interest rates, inflation, GDP — as macro context for the AI.

Free key: St. Louis Fed.

Alpha Vantage Optional Free key

Provides: US market movers (top gainers, losers, most active) as a risk-appetite signal in the AI’s context.

Free key: alphavantage.co.

AI setup

AI keys are configured in the wizard (Step 6) or anytime via Settings → AI.

Claude API key (required)

Powers all trading decisions — evaluates signals, explains reasoning, and manages risk. Get a key at console.anthropic.com. Without this key Xaltrion cannot make AI-powered decisions.

Perplexity API key (optional)

Adds real-time web market context on top of your data feeds. Get a key at perplexity.ai. Without it Xaltrion runs in Claude-only mode, which is still fully functional.

Model selection

Choose your Claude model in Settings → AI:

  • Claude Haiku 4.5 — faster, more cost-effective. Recommended for high-frequency trading.
  • Claude Sonnet 4.6 — more capable, higher cost. Recommended for complex multi-asset strategies.

AI assistant (Veylor)

Veylor is the built-in AI assistant, available from the ✨ button in any view. It knows your live setup — brokers, balances, holdings, watchlist, and running costs — so you can ask things like “why didn't it trade today?”, “what should I keep an eye on?”, or “how do I reduce my AI cost?” The estimated cost of each chat is shown, so there are no surprises.

Standard vs Full experience

By default Veylor is a read-only advisor — it answers questions but changes nothing. You can opt in to Full experience mode (Settings → AI), which lets Veylor also act on your behalf: apply configuration (tradable symbols, watchlist, sizing), run diagnostics, self-heal common issues, and — with your permission — send feedback to the team. Every action asks for your confirmation first.

Cost note: Full experience uses the AI more, so it costs more. Xaltrion makes it an explicit opt-in and shows the projected spend before you turn it on.

Cost transparency

The Operator view's AI Spend tile shows your projected cost per day and per month at your current settings. Cost scales with how much you ask the AI to do — the number of symbols, how often it runs, and whether optional research (Perplexity) is enabled. You stay in control of the spend.

Privacy

Veylor uses your own AI keys and your trading data stays on your machine. Sending feedback to the team is always opt-in.

Dashboard guide

The portal opens at http://localhost:8080 after starting the app. Choose your view from the top ribbon.

  • Simple — equity curve, open positions, recent decisions. Clean and calm.
  • Operator — multi-broker portfolio, AI spend tile, risk guardrail bars, equity graph with range selector.
  • Expert — full console, AI reasoning log, live feeds, orders, backtest, heartbeat.
  • Settings — brokers, data feeds, AI provider, risk limits, notifications, licensing.
  • Veylor assistant — the ✨ button (any view) opens the AI assistant for questions, setup help, and — in Full experience — hands-on changes.

Troubleshooting

Windows SmartScreen blocks installer

Click More info then Run anyway. The installer is unsigned during beta.

Setup wizard not appearing on first launch

Go to Settings → General → Resume onboarding to restart the setup wizard.

License token cache stale

Go to Settings → System → Clear activation cache, or delete .activation_cache.json and restart Xaltrion.

Machine binding mismatch

Each licence activates on one machine. If you’ve reinstalled, changed hardware, or moved machines, email support@xaltrion.ai to release the previous activation — then re-activate. A stale local cache can also cause this: Settings → System → Clear activation cache and restart.

Port 8080 already in use

Close any other app using port 8080 and restart Xaltrion.

Perplexity not working

Check your API key is saved in Settings → AI → Perplexity. Without a valid key, Xaltrion runs in Claude-only mode.

Reset portal layout

Use Settings → System → Reset portal layout to restore the default layout.

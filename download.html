System requirements

  • Windows 10/11 (64-bit)
  • 4 GB RAM minimum
  • Stable internet connection

Setup steps

  1. Download the ZIP package from the latest release.
  2. Extract the archive to a folder on your machine.
  3. Run XaltrionAI.exe.
  4. Enter your licence key when prompted.

Windows may show a warning - click More info, then Run anyway.
